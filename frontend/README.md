# Frontend Skeleton

This directory will hold the exam-required dApp.

Minimum UI features to implement:
- wallet connection
- wrong-network detection
- asset token balance
- governance voting power
- delegate address
- protocol-specific state
- deposit action
- swap action
- vote action
- proposal state page
- one page backed by The Graph data

Recommended stack:
- React
- Viem
- Wagmi
- Vite
